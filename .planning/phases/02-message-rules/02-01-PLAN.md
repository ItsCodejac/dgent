<task type="auto">
  <name>Task 2: Implement strip-trailers rule with fixtures</name>
  <files>src/rules/strip-trailers.ts, src/rules/index.ts, test/fixtures/strip-trailers/*.txt</files>
  <action>
    Create src/rules/strip-trailers.ts:
    - Implements the Rule interface
    - name: "strip-trailers", phase: "commit-msg", type: "fix", defaultEnabled: true
    - apply(input): removes lines matching known AI trailer patterns:
      - `Co-Authored-By:` (case-insensitive)
      - `Generated-By:`
      - `Assisted-By:`
      - `Co-authored-by:` with known AI email patterns (noreply@anthropic.com, github-copilot[bot], etc.)
      - Lines matching `🤖 Generated with *` pattern
    - Also removes trailing blank lines left by trailer removal
    - Returns the cleaned message with changed=true if anything was removed

    Register in src/rules/index.ts rules array.

    Create fixtures:
    - basic-claude-trailer.input.txt / .expected.txt — single Co-Authored-By trailer
    - multiple-trailers.input.txt / .expected.txt — multiple different trailers
    - mixed-content.input.txt / .expected.txt — trailers mixed with real content
    - false-positive-human-coauthor.input.txt / .expected.txt — Co-Authored-By with a real human email (should NOT be stripped, expected = input)
    - no-trailers.input.txt / .expected.txt — message with no trailers (expected = input, no change)
    - generated-with-footer.input.txt / .expected.txt — "🤖 Generated with Claude Code" footer line

    Run `dgent test --rule strip-trailers` to verify all pass.
  </action>
  <verify>`dgent test --rule strip-trailers` passes all fixtures</verify>
  <done>strip-trailers rule removes AI trailers, preserves human co-authors, all fixtures pass including false positive cases</done>
